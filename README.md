# Coffee-Store-Data-Analysis (Interactive Dashboard using MS Excel)
I developed a comprehensive project in Excel, creating a dashboard to visualize the data of a coffee shop. Synthesized key business outcomes to help strategic decisions. Used XLOOKUP, INDEX, MATCH, nested IF functions among others to analyze Total Sales Over Time, Sales By Country and Top 5 Customers. Timelines and multiple Slicers help for the visualization and business outcome analysis.
